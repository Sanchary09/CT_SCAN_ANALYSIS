# CT Image Processing using DICOM and NIFTI

## Overview

This project focuses on understanding and processing Computed Tomography (CT) data using DICOM and NIFTI formats. The implementation covers reading volumetric CT data, format conversion, multi-planar visualization, reconstruction, and preprocessing techniques such as windowing and normalization.

The work demonstrates a complete medical image processing pipeline using Python and visualization tools.

---

## Objectives

- Read CT data from DICOM format
- Convert DICOM to NIFTI and vice versa
- Visualize CT data in axial, sagittal, and coronal planes
- Reconstruct CT volume from slices
- Validate reconstruction accuracy
- Apply windowing (WL=50, WW=450)
- Normalize CT data for preprocessing
